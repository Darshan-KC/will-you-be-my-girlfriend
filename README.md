# 💕 Will You Be My Girlfriend?

A fun, interactive web application that asks someone to be your girlfriend in a creative and playful way! 

## ✨ Features

- **Animated User Interface**: Beautiful gradient background with floating heart animations
- **Evasive "No" Button**: The "No" button cleverly swaps positions with the "Yes" button, making it impossible (or at least very difficult) to click
- **Smooth Transitions**: Glassmorphism design with smooth animations and transitions
- **Celebration Display**: When "Yes" is clicked, a celebratory GIF appears
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Keyboard Prevention**: Tab key is disabled to prevent cheating!

## 🎯 How to Use

1. Open `index.html` in your web browser
2. Read the romantic message and question
3. Click "Yes" to see the celebration!
4. (The "No" button will playfully dodge your cursor)

## 🎨 Customization

You can easily customize this project:

- **Change the Message**: Edit the `<h1>` and `<h2>` text in `index.html`
- **Modify Colors**: Update the gradient colors in the `<style>` tag
- **Change the Celebration GIF**: Replace the GIF URL in the `gif-container` element
- **Adjust Animation Speed**: Modify the animation values in `style.css` (e.g., `8s ease-in-out`)

## 🚀 Getting Started

Simply clone or download this repository and open `index.html` in your favorite web browser. No installation or dependencies required!

```bash
# Open in your browser
open index.html          # macOS
start index.html        # Windows
xdg-open index.html     # Linux
```
